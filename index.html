UROP: Using Python in Composite Materials


In this Undergraduate Research Opportunity Program, I was involved in engineering cross-sectional diagrams for carbon fibre composites that will be used for further research.
Other than producing the cross-sectional diagram, I also made use of statistical knowledge to make sure that the cross-sectional diagram is random enough to be used generally.

UROP: Data Mining


In this Undergraduate Research Opportunity Program, I was tasked to develop a search engine based off Wikipedia and package data at the backend using JavaScript and Wikipedia API.
The search engine reccomends related words for users to explore further on top of providing short concised information on the search. It also records the searches done by the users for referencing if needed.

Personal/ Other Coding Projects

Webscraping Data Online


I have been exploring tools for webscraping since I learned how to code. Webscraping made me appreciate the wonders of coding and how useful automation can be.
Some of the webscraping libraries used were Selenium and BeautifulSoup and I have coded programs to automate collection of data, such as retrieving stock prices from the Stock Market and perform simple daily tasks that requires signing in online.

@PositiveMeBot Telegram Bot


After having a terrible term 2 experience in my first year, I decided to create a telegram bot that aims to initiate positive thinking.
The bot receives the message sent and echo the message without the negative words. You can also get a random motivational quote from the bot too!

Expenses Tracking App of Sort


This was my first personal project. I created a desktop application to track my expenses. In this project, I used Python and Kivy to build the application. The project was initially meant to be made into a phone application, but I was met with difficulties using a virtual machine. Therefore I made some adjustments and it became a desktop application instead.
Even though the project did not turn out as I expected, the project introduced me to virtual machine and also taught me how to adjust based on circumstances in coding.
